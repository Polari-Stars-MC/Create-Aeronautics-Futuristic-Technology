# Create-Aeronautics-Futuristic-Technology

Create Aeronautics: Futuristic Technology

## Current Features

- Physics Staff delete subLevel
- Physics Staff adjust subLevel

## 项目定位

- 这是一个面向 Create Aeronautics 风格的未来科技扩展模组。
- 当前阶段主要在搭建物理交互基础能力，并逐步加入自定义未来科技方块。
- 现有功能以 Physics Staff 的辅助操作为主，包括拖拽物体对正和删除。

## 首发物理方块清单

以下是推荐的第一批未来科技类物理方块，目标是先做出一套可以真正运行的基础飞行平台。

### 1. 能量核心

- 作用：为整台装置提供基础能源。
- 定位：所有未来科技物理方块的供能起点，没有它，高级模块无法稳定工作。

### 2. 小型矢量推进器

- 作用：提供定向推力，可用于前进、上升和转向。
- 定位：首发阶段最核心的动力方块，用来建立飞行与移动能力。

### 3. 姿态稳定器

- 作用：抑制翻滚、偏航和俯仰，提升物理载具稳定性。
- 定位：解决“能动但难控制”的问题。

### 4. 悬浮推进器

- 作用：持续提供稳定向上的悬浮力。
- 定位：适合悬浮平台、低空飞行器和未来风格运输载具。

### 5. 飞控计算机

- 作用：统一管理推进器、稳定器和能量分配。
- 定位：整套系统的控制中枢，可扩展悬停、自动平衡和模式切换等功能。

### 6. 能量电容

- 作用：缓存能量并平滑瞬时高功耗。
- 定位：提升推进系统和飞控系统运行稳定性。

### 7. 轻质机体方块

- 作用：作为低重量结构材料，用于搭建飞行器主体。
- 定位：强调轻量化，是未来飞行器的基础结构件。

### 8. 强化装甲方块

- 作用：提供更高防护能力，但会增加整体重量。
- 定位：让玩家在强度和机动性之间做取舍。

## 推荐开发顺序

1. 能量核心
2. 小型矢量推进器
3. 轻质机体方块
4. 飞控计算机
5. 姿态稳定器
6. 悬浮推进器
7. 能量电容
8. 强化装甲方块

## 设计目标

- 先形成一套最小闭环：供能 -> 推进 -> 控制 -> 结构。
- 首发内容优先强调“未来科技感”和“物理系统联动”，而不是单纯增加装饰方块。
- 后续可以在这套基础上继续扩展反重力、护盾、导航、自动化控制等高级内容。
